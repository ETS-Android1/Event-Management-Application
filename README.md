# Event-Management-Application
•	This android application serves as a digital notice board for all the events that takes place in a university or college.\n
•	All the upcoming, ongoing events along with completed events are displayed dynamically in the app using firebase database.\n
•	Events are grouped by the club name that hosts the particular event.\n
•	Event coordinators will be given access to upload a new event and he/she notifies users(Students,faculty) with a pop up notification(through firebase cloud messaging).\n
•	Download and install "app-debug.apk" file in your android device to expereience it!(it's safe).
